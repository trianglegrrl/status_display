Support for Smoke 2010 for Mac OS X
 
 
 

Backburner is now leveraged by the new Autodesk Smoke for Mac OS X, in both standalone and render-farm deployments. In the standalone deployment, Smoke for Mac OS X uses Backburner for background I/O, including proxy generation. In the render farm deployment, Smoke for Mac OS X can submit jobs to a render farm consisting of Burn render nodes installed on Linux workstations.
