Problems Submitting Jobs
 
 
 

These problems can occur when you attempt to submit a job to the Backburner network on Linux or Windows systems.

Problem Common Cause
The Backburner Manager is unavailable when submitting jobs. An incorrect IP address or an IP address that connects to the wrong Backburner Manager, a Backburner Manager that is not running or is hung, or communication problems between the Backburner Manager and the render nodes.

See Resolving a Problem Connecting to the Backburner Manager.

An Unknown error loading application error appears in the Backburner Manager or Monitor when submitting jobs to Backburner on a Windows system. The path to Backburner is not defined in the Path environment variable. See Adding the Backburner Path to the Windows Path Environment Variable.
