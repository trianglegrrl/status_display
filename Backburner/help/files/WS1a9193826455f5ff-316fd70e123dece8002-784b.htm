Before You Begin
 
 
 

Please note the following general points:

  • User Account and Password. Depending on how Backburner has been installed and configured, you may need a user account and password to gain access to the chosen monitor.
  • Administrator Privileges. Depending on the monitor used, you may need a user account with administrator privileges to actively manage all jobs and render nodes.
  • Manager Name or IP Address. The Backburner Monitor obtains its information from the Backburner Manager. In order to connect the Backburner Monitor to a Backburner Manager, you require the Backburner Manager host name, or IP address.
NoteThe above points are dependent on how your Backburner network is set up, and the monitor type in use. For example, the Backburner Windows Monitor automatically acquires administrator privileges when it obtains control of the job queue. No special account is needed.
