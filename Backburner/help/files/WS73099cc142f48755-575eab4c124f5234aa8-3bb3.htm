Tasks Tab
 
 
 

The following table describes the information found in the Tasks tab.

Column Description
ID The task number for this task.
Status The state of the task (active, complete, waiting, error).
Server The name of the server where the task is being executed.
Start Time The time stamp at which the task was started (YYYY-MM-DD HH:MM:SS).
Elapsed Time The time duration consumed by the task (HH:MM:SS).
Last Error The last execution error message associated with this task.
