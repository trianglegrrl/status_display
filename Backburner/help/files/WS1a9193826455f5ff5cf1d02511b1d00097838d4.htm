About the Web Browser
 
 
 

The Help is best viewed using Firefox 2 or Internet Explorer 7. Javascript must be enabled.

Take advantage of browser features while using the Help:

  • Search for text within a page using Find on This Page (Ctrl+F in most browsers).
  • Revisit pages using the Back and Forward buttons (or the History list).
  • Open a new tab or window (Ctrl+T or Ctrl+N) to show the Help in more than one tab or window at the same time.
  • Change the size of the fonts the Help uses.
