Advanced Info Tab
 
 
 

The Advanced Info Tab presents the job metadata sent by the client application to Backburner. It is helpful when troubleshooting a job. In addition, some applications submit Backburner jobs with extended instructions for the processing engine. Use the Advanced Info tab to view these extended instructions.
