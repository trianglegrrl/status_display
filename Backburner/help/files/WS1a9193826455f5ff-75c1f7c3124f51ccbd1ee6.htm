Command-Job Utility
 
 
 

The Backburner command-job utility, cmdjob, lets you submit batch, executable or script files to Backburner as "custom" jobs from the command line, for execution on render nodes. Two distinct command-job adapters are installed on the render nodes to handle the custom jobs. One adapter executes jobs as the root user. The other retains the account of the user that submitted the job. See Advanced Operations.
